# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Mutation Impact and Pathogenicity Prediction project
- Python implementation with multiple ML models:
  - Random Forest
  - Support Vector Machine (SVM)
  - Logistic Regression
  - Gradient Boosting
  - Neural Network (MLP)
  - Ensemble methods
- R implementation with multiple ML models:
  - Random Forest
  - Support Vector Machine (SVM)
  - Logistic Regression
  - XGBoost
  - Ensemble methods
- Multiple sequence encoding methods:
  - One-hot encoding
  - K-mer frequency encoding
  - Numerical encoding
- Comprehensive data loading and preprocessing pipeline
- Jupyter notebooks for interactive analysis (Python and R)
- Command-line interface for both Python and R
- Model evaluation metrics (Accuracy, Precision, Recall, F1-score, ROC-AUC)
- Model persistence (save/load trained models)
- Comprehensive documentation:
  - README.md with usage instructions
  - SETUP_GUIDE.md for installation
  - CONTRIBUTING.md for contributors
  - GITHUB_SETUP.md for repository setup
- Example scripts and quick start guide
- Requirements files for Python and R
- Conda environment configuration
- Git repository initialization
- License file (MIT License)
- .gitignore and .gitattributes files

### Features
- Binary classification of mutations (pathogenic vs benign)
- Support for multiple encoding methods
- Ensemble learning for improved accuracy
- Comprehensive evaluation metrics
- Model comparison and visualization
- Cross-platform support (Windows, macOS, Linux)

### Documentation
